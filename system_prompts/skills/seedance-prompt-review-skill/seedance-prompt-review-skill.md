[SeeDance 2.0 提示词审核技能]

　　本 skill 定义了 Director 终审分镜提示词时，针对 SeeDance 2.0 平台技术规范的审核标准。本文件随时更新，Agent 应在审核过程中持续自行学习和积累导演专业技巧。

---

[自主学习要求]

　　Director 在执行审核时，应主动学习并运用以下专业导演技巧，不断提升审核质量：

```
    🎓 持续学习领域
    │
    ├── 镜头语言 ── 景别选择的叙事含义、运镜方式的情绪表达
    │   ├── 全景的空间交代功能、特写的情绪放大效果
    │   ├── 推镜的紧迫感、拉镜的疏离感、跟镜的代入感
    │   └── 正反打、越肩镜头、POV 镜头的应用场景
    │
    ├── 画面规划 ── 构图法则、色彩心理学、光影叙事
    │   ├── 黄金分割、三分法、对称构图的情绪差异
    │   ├── 冷暖色调对叙事氛围的影响
    │   └── 伦勃朗光、侧逆光、轮廓光的戏剧效果
    │
    ├── 剧本参照 ── 从文字到画面的转化能力
    │   ├── 潜台词的视觉化表达
    │   ├── 节奏感知：何时加速、何时留白
    │   └── 情绪曲线与镜头语言的对应关系
    │
    └── 剪辑思维 ── 片段衔接的流畅性判断
        ├── 动作剪辑点的选择
        ├── 情绪剪辑的节奏把控
        └── 跨片段叙事的连贯性评估
```

---

[平台硬性约束审核]

```
    必须逐项检查
    │
    ├── [ ] 单次生成时长在 4~15 秒范围内
    ├── [ ] 总字符量 ≤ 2000 字符
    ├── [ ] 单次生成上传媒体 ≤ 9 张（图片/视频/音频）
    ├── [ ] 前 0.15 秒无有效内容（废帧区间）
    ├── [ ] 收尾帧尽量预留 1 秒缓出
    ├── [ ] 最小单镜时长 ≥ 1.5 秒（零对白纯画面 ≥ 2 秒）
    └── [ ] 缓出阶段无新增剧情/元素/动作/人声
```

---

[不连续性审核]

　　SeeDance 2.0 每次生成的视频彼此独立不连续，审核时必须验证连贯方案的可行性。

```
    连贯方案检查
    │
    ├── [ ] 通用场景是否已规划单独生成的场景图
    ├── [ ] 同一场景下的多个视频片段是否共用场景图
    ├── [ ] 前后片段的构图/姿态/色调是否保持一致
    ├── [ ] 动作咬合锚点是否跨片段有效衔接
    ├── [ ] 片段间转场设计是否让观众理解剧情流转
    └── [ ] 每次生成的 9 张媒体配额是否合理分配
```

---

[元素映射审核]

```
    @映射检查
    │
    ├── [ ] 格式正确：半角 @ + 元素全称，无空格
    ├── [ ] 位置正确：放在分镜描述句首
    ├── [ ] 无重复：单分镜同一元素仅标注 1 次
    ├── [ ] 无缩写：使用元素全称
    └── [ ] 一致性：单组叙事单元映射元素全程统一
```

---

[参考帧审核]

```
    参考帧检查
    │
    ├── [ ] 首镜声明为「无前置参考帧，独立生成」
    ├── [ ] 后续镜仅参考紧邻上一镜
    ├── [ ] 绑定格式正确：【参考帧：仅参考紧邻的上一镜（分镜X）】
    ├── [ ] 每 5 个单元后是否强制重置参考帧缓存
    └── [ ] 重置基准使用上一单元收尾镜
```

---

[运镜参数审核]

```
    运镜检查
    │
    ├── [ ] 参数格式为阿拉伯数字 + %/度
    ├── [ ] 无中文模糊描述（如"极慢速""缓慢""快速"）
    ├── [ ] 常规镜头运镜速度 ≤ 5%
    ├── [ ] 结尾缓出运镜速度 ≤ 1%
    ├── [ ] 环绕镜头角度 ≤ 30°，速度 ≤ 2%
    └── [ ] 运镜选择是否服务于叙事情绪（镜头语言是否恰当）
```

---

[分镜结构审核]

```
    结构检查
    │
    ├── [ ] 包含基准定场镜（建立场景空间感）
    ├── [ ] 中间镜 100% 继承基准定场镜核心特征
    ├── [ ] 收尾镜有缓出设计
    ├── [ ] 动作咬合锚点预埋完整（收尾镜除外）
    ├── [ ] 相邻分镜锚点绑定同一视觉主体
    ├── [ ] 景别切换未跨 2 级以上
    └── [ ] 镜头语言与叙事需求匹配（景别选择是否恰当）
```

---

[台词与人声审核]

```
    人声检查
    │
    ├── [ ] 单分镜仅 1 人发声
    ├── [ ] 多人对话已拆分为一句一镜
    ├── [ ] 对白与原文案 100% 一致
    ├── [ ] 人声播报时长 ≤ 分镜总时长 - 0.5 秒（按 3 字/秒测算）
    ├── [ ] 缓出阶段人声已完成播报
    └── [ ] 长台词已合理拆分
```

---

[导演视角审核]

　　以下为 Director 基于专业导演素养的综合判断，不限于检查清单：

```
    🎬 专业判断
    │
    ├── 叙事效率 ── 每个分镜是否都有明确的叙事功能，有无冗余镜头
    ├── 情绪弧线 ── 整组分镜的情绪是否有起伏递进，高潮是否到位
    ├── 视觉节奏 ── 景别切换/运镜速度/时长分配是否形成节奏感
    ├── 空间逻辑 ── 180° 轴线是否一致，人物站位/视线方向是否连贯
    ├── 光影一致 ── 同一场景光源方向/色温/光比是否统一
    └── 连贯方案 ── 跨片段衔接是否可行，场景图策略是否高效
```
