[SeeDance 2.0 分镜编写技能]

　　本 skill 定义了 SeeDance 2.0（即梦平台）动态视频分镜提示词的编写规范。完整技术标准参考源文件：`漫剧/即梦提示词.md`（随时更新）。

---

[平台核心限制 ⚠️]

```
    ⚠️ 重中之重
    │
    ├── 🔴 生成时长 ── 单次生成 4~15 秒视频（硬性约束，不可突破）
    ├── 🔴 每次生成不连续 ── 每次生成的视频彼此独立，无法自动衔接！
    │   ├── 必须通过可控部分（构图、姿态、场景、色调等）制造视觉连贯感
    │   ├── 通用场景需单独生成场景图，一张场景图可能贯穿多个视频片段
    │   ├── 具体连贯方案由 Director 规划，各 Agent 协作执行
    │   └── 📌 这是 SeeDance 2.0 最核心的平台约束，所有分镜设计必须围绕此限制展开
    │
    ├── 🔴 媒体上限 ── 单次生成最多上传 9 张媒体（图片/视频/音频）
    ├── 🔴 废帧 ── 前 0.15 秒为废帧，不可承载有效内容
    └── 🔴 收尾帧 ── 尽量保证最后 1 秒为缓出时长，缓出阶段无新增剧情/元素/动作/人声
```

---

[基础限制]

```
    硬性约束
    │
    ├── 语言 ── 中文
    ├── 总字符量 ── ≤ 2000 字符
    └── 最小单镜时长 ── ≥ 1.5 秒（零对白纯画面 ≥ 2 秒）
```

```
    Director 主导项（非硬性，由 Director 根据剧情决定）
    │
    ├── 分镜单元结构 ── 镜数、时长分配
    ├── 单分镜描述长度 ── 在总字符量内由 Director 分配
    ├── 景别切换方案 ── 逐级递进或特殊处理
    └── 转场方式选择 ── 淡入淡出/叠化/闪回/切黑等
```

[分镜单元结构]

　　以下为标准结构框架，具体镜数和时长分配由 Director 根据剧情节奏决定。

```
    单元组成
    │
    ├── 基准定场镜（首镜）
    │   ├── 建立场景空间感
    │   ├── 无前置参考帧，独立生成
    │   ├── 通常为全景/远景
    │   └── 长台词单元中必须为主发声分镜
    │
    ├── 动作承接镜 / 调度辅助镜（中间镜）
    │   ├── 推进动作和情节
    │   ├── 仅参考紧邻上一镜
    │   ├── 必须包含动作咬合预埋锚点
    │   └── 100% 继承基准定场镜的全部核心特征
    │
    └── 单元收尾镜（末镜）
        ├── 收束当前单元
        ├── 最后 1 秒为缓出时长（尽量保证）
        ├── 运镜速度 ≤ 1%
        └── 缓出阶段无新增剧情/元素/动作/人声
```

[元素映射规则（@映射）]

```
    格式要求
    │
    ├── 格式 ── 半角 @ + 元素全称，连续无空格
    │   ├── ✅ @林峰
    │   ├── ✅ @天枢城中央广场
    │   ├── ❌ @ 林峰（有空格）
    │   └── ❌ @林（缩写）
    │
    ├── 顺序 ── 核心人物 > 核心场景 > 核心物品
    ├── 位置 ── 放在分镜主体描述的句首
    ├── 限制 ── 单分镜同一元素仅句首标注 1 次
    └── 一致性 ── 单组叙事单元映射元素全程统一
```

---

[参考帧规则]

```
    参考帧机制
    │
    ├── 首镜 ── 无参考帧，独立生成
    ├── 后续镜 ── 仅参考紧邻上一镜
    ├── 绑定格式 ── 【参考帧：仅参考紧邻的上一镜（分镜X）】
    ├── 强制重置 ── 每 5 个单元后强制重置参考帧缓存
    └── 重置基准 ── 上一单元收尾镜作为新基准定场镜
```

---

[动作咬合预埋规则]

```
    咬合锚点
    │
    ├── 每个镜头结尾预埋 1 个可量化、可视觉化的物理动作锚点
    ├── 相邻分镜锚点必须绑定同一视觉主体
    ├── 下一镜开头帧级精准咬合该锚点
    ├── 🚫 禁止使用抽象情绪、黑场、空白画面作为锚点
    └── 收尾镜无需预埋锚点
```

---

[运镜参数]

```
    运镜控制
    │
    ├── 常规镜头 ── 运镜速度 ≤ 5%
    ├── 快节奏 ── 单镜运镜速度 ≤ 3%
    ├── 单长镜头 ── 运镜速度 ≤ 2%
    ├── 结尾缓出 ── 运镜速度 ≤ 1%（仅固定/极慢速推拉摇）
    ├── 环绕镜头 ── 角度 ≤ 30°，速度 ≤ 2%
    ├── 参数格式 ── 阿拉伯数字 + %/度（如：推轨 3%、横摇 2%）
    └── 🚫 禁止中文模糊描述（如"极慢速""缓慢""快速"）
```

[镜头转场与衔接]

　　具体镜头转场方案以 Director 审核指示为主。以下为 SeeDance 2.0 平台支持的合规转场和基础衔接规则：

```
    转场与衔接
    │
    ├── 合规转场类型（仅限以下 4 种）
    │   ├── 淡入淡出
    │   ├── 叠化
    │   ├── 闪回
    │   └── 切黑
    │
    ├── 衔接匹配方式
    │   ├── 动作匹配
    │   ├── 视线匹配
    │   ├── 色彩匹配
    │   └── 声音桥接
    │
    ├── 景别切换 ── 相邻分镜景别切换禁跨 2 级以上，逐级递进/拉回
    └── 📌 Director 对转场/景别/轴线有具体指示时，以 Director 为准
```

[台词与人声规则]

```
    人声控制
    │
    ├── 单分镜仅 1 人发声
    ├── 多人对话 ── 一句一镜
    ├── 对白 100% 来自原文案，一字不差
    ├── 语速按每秒 3 字测算
    ├── 人声播报时长 ≤ 分镜总时长 - 0.5 秒
    ├── 缓出阶段人声必须已完成播报
    └── 长台词（>5 秒或 >30 字）── 拆分为 1 主发声镜 + 辅助镜（由 Director 决定数量）
```

---

[人物造型合规规则]

```
    造型安全约束
    │
    ├── 🚫 禁止内衣/泳装类描述（即使角色设定为换装场景）
    ├── 🚫 运动服避免紧身暴露描述，使用宽松运动装替代
    ├── ⚠️ 医学/解剖类场景需额外合规审查，标注风险等级
    ├── 🚫 禁止软色情/成人化造型描述
    ├── 🚫 未成年角色严禁任何暴露性描述
    └── ✅ 人物造型描述以服装款式/材质/颜色为主，避免身体轮廓描写
```

[十重核心铁律]

```
    铁律清单
    │
    ├── 1. 台词与人声 ── 单分镜仅 1 人发声，多人对话一句一镜
    ├── 2. 音频 ── 合规音效库，单个音效仅绑定当前分镜
    ├── 3. 叙事时序锁死 ── 严格遵循文案行文顺序，禁止超前
    ├── 4. 镜头衔接与轴线 ── 以 Director 指示为主，遵循 180° 轴线
    ├── 5. 全片元素一致性 ── 人物/场景/光影/色调全程锁死
    ├── 6. SSS 电影级视觉统一 ── 运镜、布光、特效、画质标准
    ├── 7. 运镜参数精准控制 ── 量化格式标注
    ├── 8. 视觉呈现合规禁令 ── 无字幕/水印、无画面崩坏、无穿帮
    ├── 9. 分镜时长精准分配 ── 先预留缓出，再分配有效时长
    └── 10. 长台词多维度调度 ── 超 5 秒或 30 字拆分，保持人声连贯
```

[单分镜输出模板]

　　以下为完整的单分镜输出格式。每个字段独立成行，层次清晰，方便逐项审核和局部修改。

```
=== 分镜 X ===

    ┌─ 镜头信息 ──────────────────────────────────────────┐
    │  类型：【基准定场镜 / 动作承接镜X / 单元收尾镜】     │
    │  景别：全景 / 中全景 / 中景 / 近景 / 特写             │
    │  焦距：XXmm + 运镜方式（速度参数 X%）                 │
    │  时长：Xs（收尾镜标注：含 1s 缓出）                   │
    └──────────────────────────────────────────────────────┘

    ┌─ 叙事锚点 ──────────────────────────────────────────┐
    │  文案时序锚点：XXX                                    │
    │  叙事情绪锚点：XXX                                    │
    └──────────────────────────────────────────────────────┘

    ┌─ 衔接控制 ──────────────────────────────────────────┐
    │  参考帧：无前置参考帧，独立生成                       │
    │          / 仅参考紧邻的上一镜（分镜X）                │
    │  动作咬合预埋锚点：XXX                                │
    │  （收尾镜附加）缓出规范：固定镜头（速度 0%），        │
    │          核心特征全程锁死，音频自然淡化收尾            │
    └──────────────────────────────────────────────────────┘

    ┌─ 元素映射 ──────────────────────────────────────────┐
    │  @人物 @场景 @物品                                    │
    └──────────────────────────────────────────────────────┘

    ┌─ 镜头描述 ──────────────────────────────────────────┐
    │  运镜描述 + 布光描述 + 构图描述 + 画质声明            │
    │  人物动作与画面内容正文                                │
    └──────────────────────────────────────────────────────┘

    ┌─ 声音标注 ──────────────────────────────────────────┐
    │  对白：@角色 + (语气标注)说："台词"                   │
    │  音效：XXX                                            │
    │  BGM：XXX / 无                                        │
    └──────────────────────────────────────────────────────┘
```
