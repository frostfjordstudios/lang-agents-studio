[制片评分技能 — 多角色加权评分]

本 skill 定义了最终产出（分镜提示词）的多角色加权评分机制。
每个 Agent 从自己的专业视角对最终产出评分，各维度按角色专业度分配权重。

---

[评分维度（7维）]

```
    维度（满分 10 分/项）
    │
    ├── 叙事节奏 ── 节拍密度是否合理，信息量是否匹配时长
    ├── 画面感   ── 视觉描述是否具象可执行，能否形成有效画面
    ├── 动作链   ── 角色动作是否连贯，咬合锚点是否有效衔接
    ├── 镜头     ── 景别/运镜/焦距设计是否专业，节奏是否得当
    ├── 光影     ── 布光/色调是否统一，是否服务于叙事情绪
    ├── 声音     ── 配音/音效/配乐是否准确匹配画面和情绪
    └── 衔接     ── 片段间能否通过场景图等方法衔接成连贯剧情
```

---

[角色权重矩阵]

每个 Agent 对 7 个维度的评分权重不同。本职领域权重高，跨领域权重低。

```
              叙事  画面  动作  镜头  光影  声音  衔接
    导演       0.20  0.15  0.15  0.25  0.15  0.05  0.25
    编剧       0.30  0.05  0.15  0.05  0.05  0.05  0.15
    美术       0.05  0.30  0.10  0.10  0.25  0.00  0.10
    声音       0.05  0.05  0.05  0.05  0.05  0.50  0.05
    分镜       0.15  0.20  0.20  0.20  0.15  0.05  0.25
    制片       0.25  0.10  0.15  0.10  0.10  0.10  0.15
    ─────────────────────────────────────────────────────
    合计       1.00  0.85  0.80  0.75  0.75  0.75  0.95
```

注：合计不为 1.0 是正常的，最终按加权平均计算。

[每位 Agent 的评分指令]

你是 {agent_name}，请从你的专业视角对最终分镜提示词进行评分。

评分时：
1. 对 7 个维度逐一打分（0-10 分）
2. 在你擅长的维度上要严格、专业、挑剔
3. 在非本职维度上给出你作为旁观者的直觉评分
4. 用 1-2 句话点评每个维度
5. 指出具体问题和改进建议

[导演交叉验证]

导演评分时有额外职责：核查各 Agent 产出是否忠实于导演最初的拆解指令。

```
    导演交叉验证项
    │
    ├── 美术产出 vs 导演拆解中的视觉指令 → 是否偏差/曲解
    ├── 声音产出 vs 导演拆解中的声音指令 → 是否偏差/曲解
    ├── 分镜产出 vs 导演拆解中的运镜指令 → 是否偏差/曲解
    └── 如有偏差，指出：原始指令 vs 实际产出 vs 改进方向
```

导演评分输出额外包含：

```
🔍 导演交叉验证:
  美术忠实度: X/10 — [是否偏离拆解指令]
  声音忠实度: X/10 — [是否偏离拆解指令]
  分镜忠实度: X/10 — [是否偏离拆解指令]
  ⚠️ 偏差详情:（如有）
    - [原始指令] → [实际产出] → [需纠正]
```
