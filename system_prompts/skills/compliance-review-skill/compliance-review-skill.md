[合规审核技能]

　　本 skill 定义了 Director 执行三步审核中「合规审核」环节的详细标准。规范内容参考 `漫剧/即梦提示词.md`，适用于剧本审核和分镜终审两个阶段。

---

[绝对禁止生成内容]

　　以下内容触发任意一条，直接终止生成，不可修正：

```
    🔴 一级严重违规（立即终止）
    │
    ├── ❌ 违反国家法律法规、危害国家安全、破坏民族团结、宣扬封建迷信
    ├── ❌ 低俗色情、暴力血腥、恐怖惊悚内容
    ├── ❌ 虚假时政/民生/突发事件/医疗/金融/法律误导性内容
    ├── ❌ 侵害未成年人合法权益、危害未成年人身心健康的内容
    │   └── 包括未成年人吸烟饮酒、危险动作、软色情、成人化造型
    ├── ❌ 所有公开 IP 一律禁止使用
    │   ├── 未授权的真人肖像、公众人物形象
    │   ├── 影视/动漫/游戏等受版权保护的 IP 形象
    │   └── 受著作权保护的原创内容
    ├── ❌ 枪支弹药、易燃易爆品、管制器具及仿真品的特写/核心聚焦
    ├── ❌ 二维码、联系方式、网址、社交账号等引流内容
    ├── ❌ 歧视性内容、仇恨言论、负面价值观引导
    ├── ❌ 自杀/自残/暴力极端内容
    └── ❌ 其他违背公序良俗的内容
```

[人物造型安全约束]

```
    🛡️ 造型合规红线
    │
    ├── 🚫 禁止内衣/泳装类描述（即使角色设定为换装场景）
    ├── 🚫 运动服避免紧身暴露描述，使用宽松运动装替代
    ├── ⚠️ 医学/解剖类场景需标注风险等级，额外合规审查
    ├── 🚫 禁止软色情/成人化造型描述
    ├── 🚫 未成年角色严禁任何暴露性描述
    └── ✅ 造型描述以服装款式/材质/颜色为主，避免身体轮廓描写
```

[二级违规与自动合规替换]

　　以下内容属于可合规化调整的非核心风险，执行语义无损替换后方可继续：

```
    🟡 二级轻微违规（自动替换）
    │
    ├── 平台限流敏感词、合规边缘表述
    ├── 擦边球修饰词、网络不良热梗
    ├── 非必要违规运镜描述
    ├── 可修改的非侵权元素
    ├── 不影响核心叙事的字符超限
    └── 平台未支持的声线/特效/转场需求
```

```
    替换原则
    │
    ├── 语义无损 ── 仅修改违规词汇，核心剧情/对白/叙事逻辑 100% 保留
    ├── 合规优先 ── 替换后 100% 符合国家法规与平台规则
    ├── 透明公示 ── 最终输出附带修改公示（位置、原内容、替换内容、原因）
    └── 二次校验 ── 替换后执行二次合规终审，确认无违规方可继续
```

[风险等级标注]

```
    风险分级
    │
    ├── 🟢 低风险 ── 常规内容，无需额外处理
    │
    ├── 🟡 中风险 ── 需调整措辞或弱化描写
    │   ├── 战斗场景中的暴力动作（需控制程度）
    │   ├── 角色受伤描写（避免血腥细节）
    │   └── 情感亲密场景（需控制尺度）
    │
    └── 🔴 高风险 ── 必须修改或删除
        ├── 触发内容安全红线的描写
        ├── 可能导致模型拒绝生成的内容
        └── 医学/解剖类未经特殊审查的内容
```

[视觉呈现合规]

```
    画面合规要求
    │
    ├── 🚫 全片无任何非平台合规字幕、水印、标识性文字
    ├── 🚫 无引流类二维码/条形码/联系方式
    ├── 🚫 无画面崩坏、人物变形、主体丢失、透视错乱
    ├── 🚫 无与文案内容无关的元素
    ├── 🚫 核心主体/人物面部不得超出画面安全区或被裁切
    └── 🚫 所有公开 IP 形象、商标、受版权保护内容一律禁止出现
```

---

[敏感元素管控]

```
    敏感元素处理
    │
    ├── 烟酒、管制器具仿真品等敏感元素
    │   ├── 仅当文案明确提及且为叙事核心节点、无替代方案时
    │   ├── 方可做极致背景虚化呈现
    │   ├── 🚫 严禁特写/核心画面聚焦
    │   └── 无法虚化则直接终止生成
    │
    └── 未成年人出镜
        ├── 内容必须正向健康
        └── 面部特写单镜时长不得超过 3 秒
```

```
        ├──────┼──────────┼──────────┼──────────────────────────┤
        │  1   │ [幕/镜号]│   🟡/🔴  │ [具体问题 + 修改建议]    │
        │ ...  │   ...    │   ...    │   ...                    │
        └──────┴──────────┴──────────┴──────────────────────────┘

    造型安全检查：✅ 全部合规 / ❌ 存在问题（列表）
    公开 IP 检查：✅ 无侵权 / ❌ 存在问题（列表）
    平台规范检查：✅ 全部合规 / ❌ 存在问题（列表）
```
